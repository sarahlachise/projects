Le Gameplay

Il désigne la prise en main du jeu que l'on peut traduire en français par jouabilité. Ce terme est toutefois aussi associé à l'ensemble des sensations ressenties par le joueur lors de son expérience de jeu (plaisir, frustration,etc.)

Exemples de différents gameplays :
  • Candy Crush
  • Zelda
  • 2048


Conseil : jouer soi-même à différents types de jeux pour comprendre le ressenti
Article "Quel gameplay pour un serious game à destination des élèves ?"

Les joueurs

Quant il s'agit de jeu, ne pas perdre de vue le fait que les élèves sont des joueurs.

  • Faire un sondage pour connaître les jeux auxquels jouent chacun
  • Définir le profil de joueur de vos élèves : Le test de Bartle
  • Différencier les jeux proposés aux élèves en fonction des résultats, dans la mesure du possible
  • Adapter le jeu à l'âge de l'élève dans le cas du serious gaming (vision 3D, déplacement dans l'espace, complexité du scénario...)

La réalisation d'un serious game

Récit d'expérience : c'est très difficile.
2 tentatives :
  • Inimaginable
  • Emile en ville

Principaux problèmes rencontrés :
  • Trop de texte pour faire passer des connaissances : les élèves ne lisent quasiment jamais les consignes dans un jeu.
  • Pas assez de feedbacks : expliquer à l'élève ses erreurs
  • Problème d'outils : développer from scratch demande énormément de temps, avec Xia, possible de faire une proposition mais ne permet pas de finaliser le jeu pour le moment.
  • C'est un métier : il faut maîtriser les bases du game design sur le bout des doigts.

Pour créer un jeu, utiliser des logiciels adaptés.
